# Lesspass-powershell

> LessPass open source password manager (powershell implementation)

## Install

```powershell
PS> Install-Module -Name Lesspass
```

## Usage

```powershell
PS> LessPass
```

**Help**

```powershell
PS> LessPass -?
```
